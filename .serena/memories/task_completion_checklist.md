# Task Completion Checklist

Before marking any task complete:

1. [ ] Code follows style conventions (ruff check + format)
2. [ ] All tests pass: `make test`
3. [ ] Coverage >=80%: `make test COVERAGE=true`
4. [ ] No lint issues: `make lint-strict`
5. [ ] No hardcoded secrets (use placeholders)
6. [ ] Type hints present for new Python code
7. [ ] Conventional commit message written
8. [ ] CHANGELOG.md updated if user-facing change
9. [ ] README.md updated if API/usage changed
